## ‚öôÔ∏è Installation & Setup

To run this project locally, please follow these steps:

1.  **Prerequisites:**

    - Make sure you have **NetBeans IDE** installed.
    - Install and run **XAMPP** (Windows) or **LAMPP** (Linux).

2.  **Database Setup:**

    - Open XAMPP Control Panel and start **Apache** and **MySQL**.
    - Go to `http://localhost/phpmyadmin`.
    - Create a new database (e.g., `toko_nawir`).
    - Import the SQL file located in the `database` folder of this repository (if available).

3.  **Run the App:**
    - Open the project in NetBeans.
    - Check the database connection string in the code to ensure it matches your local MySQL config.
    - Clean and Build, then Run the project.
